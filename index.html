TERMINOS SIMPLES

DECK

El deck es donde estan todas las cartas de tu maso, o disiciendolo de manera simple es donde estan todas las cartas con que juegas.

BRICK

Este termino se utiliza por los jugadores para decir que su mano es inserbible. o no pueden hacer nada con ella es mas un termino de queja, tambien puedes decir que te salieron ladrillos en tu mano

RANK UP

este es un termino que no es tan utilizado por los jugadores, pero siempre es bueno saber. es utilizado para decribir que a subir de nivel un monstruo. ya con otro monstruo o cartas magicas y de trampa

ENCADENAR

este termino si es bastante conocido ya que pasa todo el tiempo dentro del juego, este termino se utiliza cuando el efecto de una carta es activado y otro despues, lo que se llama efecto en cadena o encadenar

ROBAR

este no es del todo un termino es mas una cacion que se lleva a cabo cada turno, al inicio de cada "main phase" puedes robar una carta o mas dependiendo si se activa algun efecto que lo permita

MANO

se le llama mano al nunmero de cartas que tu sostienes, en la partida el numero de las cartas que sostienes puede variar muchicimo por efectos de cartas, pero al inicio de la partida se inicia siempre con 5 cartas en la mano

LIFE POINTS(LP)

este termino hace pues a los que dice, o sea tu puntos de vida que tienes en cada partida estos siempre seran 8000 y si llegan a 0 perderas

FASES

las fases o phases son parte fundamental en el juego esto debido a que cada cada accion del juego ducede en una de estas existen estos tipos de fases

  • draw phase
  • main phase
  • battle phase
  • main phase 2
  • end phase
